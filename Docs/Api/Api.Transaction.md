# Domain Aggregate

## Transaction

### Create Transaction

#### Create Transaction Request

```js
POST /users/{userId}/periods/{periodId}
```

```json
{
    "transactionType": "income", // or expense
    "amount": {
        "value": 100,
        "currencyCode": "USD"
    },
    "description": "deleting db tables",
    "transactionDateTime": "2024-01-01T12:00:00.0000000Z",
}
```

#### Create Transaction Response

```js
201 OK
```
